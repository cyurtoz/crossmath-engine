# Java Console Application

This is a simple Java console application with a `Main.java` entry point.

## How to Build and Run

### Prerequisites
- Java Development Kit (JDK) 8 or higher

### Compile
```
javac src/Main.java
```

### Run
```
java -cp src Main
```
